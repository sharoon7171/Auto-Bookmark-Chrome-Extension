Keyboard Shortcut

To set a custom shortcut for manually applying rules:

  1. Go to chrome://extensions/shortcuts
  2. Find "Auto Bookmark" in the list
  3. Click on the input field next to "Apply bookmarking rules"
  4. Press your desired key combination (e.g., Ctrl+Shift+B)

Example shortcut: Ctrl+Shift+B (on Windows/Linux) or Command+Shift+B (on Mac)
